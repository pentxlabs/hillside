# Carousel Images

Place your carousel/slider items here.

## Recommended Specifications
- **Resolution:** 800x600px (4:3 aspect ratio)
- **Format:** JPG or WebP
- **File Size:** < 200KB

## Usage
- Service showcases
- Product displays
- Feature highlights
- Team members
- Partner logos

## Naming Convention
Use category prefixes: `service-oil-change.jpg`, `feature-wifi.jpg`
